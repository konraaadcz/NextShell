# NextShell Installer

Welcome to the official installer for **NextShell**! This installer will always fetch and install the latest version of the application, ensuring you stay up to date.

## Installation Guide

### Step 1: Download the Installer
Download the `.exe` file on GitHub.

### Step 2: Run the Installer
- Double-click the downloaded `.exe` file.
- Follow the on-screen instructions to complete the installation.

### Step 3: Start Using NextShell
Once the installation is complete, you can launch NextShell in the applications directory by running the `start.bat` file

---

## System Requirements

Before installing NextShell, ensure that your system meets the following requirements:

### Minimum Requirements:
- **OS:** Windows 10/11 (64-bit)
- **RAM:** 500 MB
- **Storage:** 50 MB of free space
- **Java Runtime:** Java 11 or later

### Recommended Requirements:
- **OS:** Windows 10/11 (64-bit)
- **RAM:** 1,5 GB or more
- **Storage:** 100 MB of free space
- **Java Runtime:** Latest LTS version of Java

---

## Troubleshooting & Support

If you encounter any issues during installation, try the following:

### Common Issues:

#### 1. "Windows protected your PC" warning
- If Windows Defender SmartScreen blocks the installer, click **More info** → **Run anyway**.

#### 2. Installation failed or stuck
- Make sure you have administrator rights.
- Disable antivirus software temporarily if it is blocking the installation.
- Check that you have enough disk space.

#### 3. Java-related errors
- Ensure that you have **Java 11 or later** installed. You can download the latest Java version from [Oracle](https://www.oracle.com/java/technologies/javase-downloads.html) or [Adoptium](https://adoptium.net/).

If the issue persists, please report it on the [Issues](https://github.com/konraaadcz/NextShell/issues) tab on GitHub. Include error messages and system details to help us diagnose the problem.
